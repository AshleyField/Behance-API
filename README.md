# Behance-API

This project is based around the Behance API. The goal is to produce a basic website which shows a specific users projects and allows users to look into an individual project.

###### Development Tools

- HTML
- CSS
- JavaScript
- JSONp
- Node.js
- Gulp

**Plugins Used**

- jQuery
- Underscore.js
- Swiper.js
- Moment.js
- Lightbox by Lokesh Dhakar
- Numeral.js
